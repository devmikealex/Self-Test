isNaN функция функции Number typeof проверка

Функция isNaN() определяет является ли литерал илипеременная нечисловым значением (NaN) или нет. При работе с функцией необходимо проявлять осторожность так как онаимеет свои особенности. В качестве альтернативы можно использовать методNumber.isNaN()из ECMAScript 6, или дополнительно проверять литерал или переменную нанечисловое значение при помощиtypeof.

Проверки NaN == NaN и NaN === NaN в качестве значения вернут false. Следовательно, для проверки нужна функция isNaN.

isNaN приводит входящий аргумент к числу - если приведение успешное, возвращает false, если нет - возвращает true.

Number([])=== 0

Код для проверки:

console.log(isNaN());// true
console.log(isNaN(undefined)); // true
console.log(isNaN({})); // true
console.log(isNaN([])); // false
