XML XML разметка расширяемый язык разметки %% Extensible Markup Language

XML - это «расширяемый язык разметки» подобный HTML. Является рекомендацией сообщества W3C в качестве языка разметки общего назначения. В отличии от остальных языков разметки, XML сам по себе не определён - вы должны сами определять используемые теги. Основной целью XML является передача данных между разными системами такими как интернет.

XML разрабатывался как язык с простым формальным синтаксисом, удобный для создания и обработки документов как программами, так и человеком, с акцентом наиспользование в Интернете.

Много языков базируются на XML. Некоторые примеры: XHTML, MathML, SVG, XUL, XBL, RSS, и RDF.
