Стрелочные функции функция this JavaScript %% Arrow function expressions

Каждая функция в JavaScript определяет свой собственный контекст this, но внутри стрелочных функций значение this то же самое, что и снаружи. Стрелочные функции не имеют своего this. 

const sum = (a, b) => {
  const result = a + b;
  return result;
};
