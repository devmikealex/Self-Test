JSX JSX расширение синтаксиса JavaScript %% JavaScript Syntax Extension / JavaScript XML

Расширение языка JavaScript JSX - это расширение синтаксиса JavaScript, которое позволяет использовать HTML-подобный синтаксис для описания структуры интерфейса. Синтаксис JSX не является ни строкой, ни HTML.

JSX является синтаксическим сахаром для функции React.createElement(component, props, ...children).
После компиляции каждое JSX-выражение становится обычным вызовом JavaScript-функции, результат которого — объект JavaScript.

Особенности:

  1. Комментарии надо писать в скобках {}
  2. Всегда закрывать одиночные теги <hr /> <App/>
  3. Стандартные теги с маленькой буквы и без тире (кроме data-)
  4. Кастомные компонены называются всегда с большой буквы
  5. Не использовать class, вместо него className
  6. Не использовать for, вместо него htmlFor
  7. Код JavaScript надо оборачивать в фигурные скобки {} (в скобках только выражения?)
