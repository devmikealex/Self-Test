HTML HTML Hypertext Markup Language разметка гипертекст %% HyperText Markup Language

HTML - это код, который используется для структурирования и отображения веб-страницы и её контента. HTML не является языком программирования, это язык гипертекстовой разметки документов, и используется, чтобы сообщать браузеру, как отображать веб-страницы, которые вы посещаете.

Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS или открывают с локального диска, далее интерпретируют код в интерфейс, который будет отображаться на экране монитора.
